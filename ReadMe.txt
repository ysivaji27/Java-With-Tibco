Project Title
Java Program to send and receive Messages from Tibco server.

Getting Started:
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 


Prerequisites:-
 1. jdk1.8
 2. Install Tibco Server.
 3. Eclipse 

 Copy below jar file from tibco server to java project
  1. javax.jms-api-2.0.jar
  2. tibemsd_sec.jar
  3. tibjms.jar
  4. tibjmsadmin.jar
  5. tibjmsapps.jar
  6. tibjmsufo.jar
  7. tibrvjms.jar
  
 Now you can tun Sender and receiver programs to send and receive messages
